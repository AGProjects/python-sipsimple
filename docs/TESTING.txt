
See the INSTALL files for how to install the software on your platform. To
test the software you may use the provided sample clients:

sip_register
sip_rtp_audio_session
sip_msrp_im_session
sip_msrp_file_transfer
sip_presence_publish
sip_presence_subscribe

Running each script with --help argument displays the script purpose and
supported or required arguments.

First create a configuration file in your home directory called pypjua.ini
This configuration file must contain the SIP credentials. Example:

[Account]
username = alice
domain = example.com
password = 1234
xcap_root=https://xcap.example.com/xcap-root/

If you do not have a SIP account you may register one for free at:
http://sip2sip.info
